# java_gui_curd
一个大作业项目，使用java原生gui对数据库进行增删查改

**摘  要** 

这是一个基于javaGUI界面和mysql数据库做的一个交易买卖系统，该系统能实现如下功能： 

- 通过不同的面板选项登录会跳转到不同的面板，看到不同的内容。
- 用户可以在对应面板发布出售信息
- 用户可以对面板的各类出售信息执行购买操作
- 刷新获取最新价格

该系统使用MySQL5.0.67创建数据库，使用Eclipse Java 2019-03作为开发平台，使用的JDK版本为jdk1.8.0_211。
